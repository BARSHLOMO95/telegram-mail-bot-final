# Marketplace - קטלוג מוצרים

אתר marketplace בסגנון יופאו (Youpao) המאפשר הצגת מוצרים בקטגוריות שונות עם תמונות מרובות וקישורים לחנות.

## תכונות

### ממשק משתמש
- 🏠 דף הבית עם תצוגת כל המוצרים
- 📁 סינון לפי קטגוריות
- 🖼️ הצגת תמונות מרובות לכל מוצר
- 🛒 קישורים ישירים לרכישה בחנות
- 📱 עיצוב רספונסיבי (מותאם לנייד)
- 🎨 ממשק מודרני ואסתטי

### ממשק ניהול
- ➕ הוספת/עריכת/מחיקת קטגוריות
- 📦 הוספת/עריכת/מחיקת מוצרים
- 🖼️ העלאת תמונות מרובות לכל מוצר
- 🗑️ מחיקת תמונות
- 💰 קביעת מחירים
- 🔗 הוספת קישורים לחנויות

## התקנה והפעלה

### דרישות מקדימות
- Python 3.8 ומעלה
- pip (מנהל חבילות Python)

### שלבי התקנה

1. **עבור לתיקיית הפרויקט**
```bash
cd marketplace
```

2. **התקן את התלויות**
```bash
pip install -r requirements.txt
```

3. **הפעל את השרת**
```bash
python app.py
```

4. **פתח את הדפדפן**
- **ממשק משתמש**: http://localhost:5000
- **ממשק ניהול**: http://localhost:5000/admin

## מבנה הפרויקט

```
marketplace/
├── app.py                  # שרת Flask ראשי
├── database.py            # ניהול מסד נתונים SQLite
├── marketplace.db         # מסד נתונים (נוצר אוטומטית)
├── requirements.txt       # תלויות Python
├── static/
│   ├── css/              # קבצי עיצוב (אם צריך)
│   ├── js/
│   │   └── admin.js      # לוגיקה של ממשק ניהול
│   └── uploads/          # תמונות שהועלו
└── templates/
    ├── index.html        # דף הבית (ממשק משתמש)
    ├── admin.html        # ממשק ניהול
    └── product.html      # עמוד פרטי מוצר
```

## שימוש במערכת

### הוספת קטגוריה חדשה
1. היכנס לממשק הניהול (http://localhost:5000/admin)
2. לחץ על "הוסף קטגוריה חדשה"
3. מלא שם ותיאור
4. לחץ על "שמור"

### הוספת מוצר חדש
1. היכנס לממשק הניהול
2. עבור לטאב "מוצרים"
3. לחץ על "הוסף מוצר חדש"
4. מלא את הפרטים:
   - בחר קטגוריה
   - שם המוצר
   - תיאור
   - מחיר (אופציונלי)
   - קישור לחנות (אופציונלי)
   - העלה תמונות (ניתן לבחור מספר תמונות בבת אחת)
5. לחץ על "שמור"

### עריכת מוצר קיים
1. בטאב "מוצרים" לחץ על כפתור "ערוך" במוצר הרצוי
2. ערוך את הפרטים
3. להוסיף תמונות נוספות או למחוק תמונות קיימות
4. לחץ על "שמור"

## API Endpoints

המערכת כוללת API מלא לניהול התוכן:

### קטגוריות
- `GET /api/categories` - קבלת כל הקטגוריות
- `POST /api/categories` - יצירת קטגוריה חדשה
- `PUT /api/categories/<id>` - עדכון קטגוריה
- `DELETE /api/categories/<id>` - מחיקת קטגוריה

### מוצרים
- `GET /api/products` - קבלת כל המוצרים
- `GET /api/products/<id>` - קבלת מוצר ספציפי
- `POST /api/products` - יצירת מוצר חדש
- `PUT /api/products/<id>` - עדכון מוצר
- `DELETE /api/products/<id>` - מחיקת מוצר

### תמונות
- `POST /api/products/<id>/images` - העלאת תמונות למוצר
- `DELETE /api/images/<id>` - מחיקת תמונה

## התאמה אישית

### שינוי צבעי העיצוב
ערוך את המשתנים ב-CSS:
```css
:root {
    --primary-color: #ff6b6b;    /* צבע ראשי */
    --secondary-color: #4ecdc4;  /* צבע משני */
}
```

### שינוי פורט השרת
ערוך את הקוד ב-`app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5000)  # שנה את 5000 לפורט אחר
```

### הגבלת גודל קבצים
ערוך ב-`app.py`:
```python
app.config['MAX_CONTENT_LENGTH'] = 16 * 1024 * 1024  # 16MB
```

## פריסה (Deployment)

### Heroku
1. הוסף `Procfile`:
```
web: cd marketplace && python app.py
```

2. הוסף `runtime.txt`:
```
python-3.11.0
```

3. עדכן `app.py` לקריאת PORT מהסביבה:
```python
port = int(os.environ.get('PORT', 5000))
app.run(debug=False, host='0.0.0.0', port=port)
```

### שרת לינוקס (VPS)
1. התקן nginx
2. הגדר gunicorn:
```bash
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

3. הגדר nginx reverse proxy

## תמיכה ובעיות נפוצות

### המערכת לא עולה
- ודא ש-Python מותקן: `python --version`
- ודא שכל התלויות מותקנות: `pip install -r requirements.txt`
- בדוק שהפורט 5000 פנוי

### תמונות לא נטענות
- ודא שתיקיית `static/uploads` קיימת
- בדוק הרשאות כתיבה לתיקייה

### שגיאות במסד נתונים
- מחק את `marketplace.db` והפעל מחדש - המערכת תיצור אותו אוטומטית
